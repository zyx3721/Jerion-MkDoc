# 下一代防火墙



1、设备登录方式：

（1）电脑-网线接MGMT或0号口（有MGMT的为管理口，没有写的0号口为管理口），用192.168.0.1登录，默认账号admin 密码Admin@123。（电脑配置同网段即可）

（2）Console线登录，默认账号密码与web一致。

（3）无论是web 或 console 第一次登录都需要修改密码。

（4）模拟器是必须进入命令行开启管理功能才能进web，console改了以后就无需改了。
